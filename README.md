# Пошук по великому Excel-архіву на macOS

Ця програма допомагає швидко шукати інформацію у великій кількості Excel-файлів.

Вона працює локально на вашому Mac:

- без хмари;
- без окремого сервера;
- без передачі файлів кудись в інтернет.

Програма вміє:

- знаходити файл за назвою;
- шукати текст усередині Excel;
- показувати, у якій папці лежить файл;
- показувати аркуш, рядок, колонку і комірку, де знайдено збіг.

## Найпростіший спосіб запуску

Якщо ви не хочете розбиратися з Git і командами, зробіть так:

### Крок 1. Завантажте програму

1. Відкрийте цю сторінку:
   `https://github.com/MishchenkoDaniil/excel-archive-search-macos`
2. Натисніть зелену кнопку `Code`.
3. Натисніть `Download ZIP`.
4. Дочекайтеся, поки файл завантажиться.

### Крок 2. Розпакуйте архів

1. Відкрийте папку `Downloads`.
2. Знайдіть ZIP-файл.
3. Натисніть на нього двічі.
4. Після розпакування відкрийте папку `excel-archive-search-macos`.

### Крок 3. Запустіть програму

1. У папці проєкту знайдіть файл `bootstrap.command`.
2. Натисніть на нього двічі.
3. Якщо macOS покаже попередження про безпеку:
   відкрийте `System Settings -> Privacy & Security`
4. Дозвольте запуск файлу.
5. Запустіть `bootstrap.command` ще раз.

### Крок 4. Дочекайтеся запуску

Під час першого запуску Mac:

- підготує все потрібне для роботи програми;
- встановить потрібні компоненти;
- відкриє програму у браузері.

Зазвичай програма відкривається за адресою:

```text
http://localhost:8501
```

## Якщо хочете запустити через Terminal

Цей спосіб теж простий, але потрібно вставити кілька команд.

### Як відкрити Terminal

1. Натисніть `Command + Space`
2. Введіть `Terminal`
3. Натисніть `Enter`

### Якщо ви вже завантажили ZIP

Перейдіть у папку програми, наприклад так:

```bash
cd ~/Downloads/excel-archive-search-macos
```

Потім виконайте:

```bash
chmod +x bootstrap_mac.sh bootstrap.command
./bootstrap_mac.sh
```

## Якщо хочете саме через Git

Цей варіант потрібен тільки якщо ви знаєте, що таке `git clone`, або якщо вам так зручніше.

### Крок 1. Відкрийте Terminal

1. Натисніть `Command + Space`
2. Введіть `Terminal`
3. Натисніть `Enter`

### Крок 2. Перевірте Git

Виконайте:

```bash
git --version
```

Якщо Git не встановлений, виконайте:

```bash
xcode-select --install
```

### Крок 3. Завантажте проєкт

Перейдіть у папку `Downloads`:

```bash
cd ~/Downloads
```

Потім виконайте:

```bash
git clone https://github.com/MishchenkoDaniil/excel-archive-search-macos.git
cd excel-archive-search-macos
```

### Крок 4. Запустіть програму

```bash
chmod +x bootstrap_mac.sh bootstrap.command
./bootstrap_mac.sh
```

## Як користуватися програмою

Після запуску в браузері:

1. Натисніть `Обрати у Finder`
2. Виберіть головну папку, у якій лежать ваші Excel-файли
3. Натисніть `Індексувати`
4. Дочекайтеся завершення першого сканування
5. У блоці `Пошук` введіть текст, який хочете знайти
6. Оберіть режим:
   - `За назвою файлу`
   - `За вмістом`
   - `Комбінований`
7. Перегляньте знайдені результати

## Що означають кнопки

- `Індексувати` — перше сканування файлів
- `Оновити індекс` — оновити базу, якщо файли змінились
- `Очистити застарілі записи` — прибрати з бази записи про файли, яких вже немає
- `Обрати у Finder` — вибрати папку через звичайне вікно macOS

## Що показується в результатах

Залежно від режиму пошуку програма показує:

- назву файлу;
- папку;
- повний шлях до файлу;
- аркуш;
- рядок;
- колонку;
- адресу комірки;
- значення знайденої комірки.

## Приклад папки для пошуку

Наприклад:

```text
~/Downloads/МСЕК(НАБРАНЕ)
```

## Якщо щось не запускається

### 1. Не відкривається `bootstrap.command`

Відкрийте:

`System Settings -> Privacy & Security`

І дозвольте запуск файлу.

### 2. Немає Python 3

Встановіть Python з офіційного сайту:

https://www.python.org/downloads/macos/

Після цього запустіть `bootstrap.command` ще раз.

### 3. Не працює Git

Це не проблема. Просто не використовуйте Git.

Замість цього:

- завантажте `ZIP`;
- розпакуйте;
- запустіть `bootstrap.command`.

## Для більш досвідченого користувача

Якщо ви хочете все зробити вручну:

```bash
python3 -m venv .venv
source .venv/bin/activate
python -m pip install --upgrade pip
pip install -r requirements.txt
streamlit run app.py
```

## Що є всередині проєкту

- `app.py` — інтерфейс програми
- `indexer.py` — індексація Excel-файлів
- `database.py` — локальна база SQLite
- `search.py` — пошук по базі
- `utils.py` — допоміжні функції
- `bootstrap_mac.sh` — запуск через Terminal
- `bootstrap.command` — запуск подвійним кліком у Finder
