# Пошук по великому Excel-архіву на macOS

Локальний застосунок на Python для індексації та швидкого пошуку по `.xlsx` і `.xlsm` файлах у вкладених папках. Усе працює локально на Mac: без хмари, без окремого сервера, лише `Streamlit + SQLite`.

## Можливості

- рекурсивна індексація всіх Excel-файлів у вибраній папці;
- пошук за частиною назви файлу;
- пошук за текстом усередині Excel;
- комбінований пошук із фільтрами по папці та імені файлу;
- показ шляху до файлу, аркуша, рядка, колонки й адреси комірки;
- пропуск файлів, які не змінювалися;
- перевидалення старих записів, якщо файл змінився;
- очищення "мертвих" записів, якщо файл видалили з диска;
- експорт результатів пошуку в Excel;
- відкриття файлу або його папки у Finder;
- логування помилок без зупинки всієї індексації.

## Структура проєкту

- `app.py` — локальний інтерфейс Streamlit
- `indexer.py` — індексація Excel-файлів
- `database.py` — схема та робота з SQLite
- `search.py` — SQL-пошук по індексу
- `utils.py` — нормалізація тексту, експорт і відкриття файлів
- `bootstrap_mac.sh` — запуск із нуля через Terminal
- `bootstrap.command` — подвійний клік у Finder для запуску bootstrap
- `excel_index.db` — локальна база, створюється автоматично після першої індексації

## Швидкий старт на macOS

### Варіант 1. Найпростіший запуск

1. Завантажте або клонюйте репозиторій.
2. Відкрийте папку проєкту у Finder.
3. Запустіть файл `bootstrap.command`.
4. Дочекайтеся встановлення залежностей і автоматичного старту застосунку.

Якщо macOS попереджає про запуск скрипта, відкрийте `System Settings -> Privacy & Security` і дозвольте запуск.

### Варіант 2. Запуск через Terminal

```bash
cd excel-archive-search-macos
chmod +x bootstrap_mac.sh bootstrap.command
./bootstrap_mac.sh
```

## Встановлення з нуля вручну

Цей варіант потрібен, якщо на Mac ще немає Python 3 або ви хочете все зробити вручну.

### 1. Встановіть Python 3

Можна одним із способів:

- завантажити інсталятор з офіційного сайту: https://www.python.org/downloads/macos/
- або через Homebrew:

```bash
brew install python
```

Перевірка:

```bash
python3 --version
```

### 2. Завантажте проєкт

```bash
git clone https://github.com/MishchenkoDaniil/excel-archive-search-macos.git
cd excel-archive-search-macos
```

### 3. Створіть віртуальне оточення

```bash
python3 -m venv .venv
source .venv/bin/activate
```

### 4. Встановіть залежності

```bash
python -m pip install --upgrade pip
pip install -r requirements.txt
```

### 5. Запустіть застосунок

```bash
streamlit run app.py
```

Після запуску відкрийте у браузері адресу на кшталт:

```text
http://localhost:8501
```

## Як користуватися

1. У полі `Коренева папка з Excel-файлами` вкажіть шлях до каталогу з архівом.
2. Натисніть `Індексувати` для першого запуску.
3. Для повторного проходу натисніть `Оновити індекс`.
4. Використовуйте один із режимів пошуку:
   `За назвою файлу`, `За вмістом`, `Комбінований`.
5. За потреби додайте фільтр по папці або імені файлу.
6. Перегляньте результати, експортуйте їх у Excel або відкрийте файл/Finder прямо з інтерфейсу.

### Приклад шляху

```text
~/Downloads/МСЕК(НАБРАНЕ)
```

## Особливості

- застосунок працює повністю локально;
- використовується тільки SQLite;
- структура Excel-файлів може бути довільною;
- формули не обчислюються через Excel, читається збережене значення, доступне через `openpyxl`;
- тимчасові файли типу `~$filename.xlsx` ігноруються;
- пошук адаптований до кирилиці та Unicode-нормалізації імен файлів на macOS.
