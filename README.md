# Пошук по великому Excel-архіву на macOS

Програма працює локально на Mac і допомагає шукати інформацію у великій кількості Excel-файлів.

## Як запустити

### 1. Завантажте програму

1. Відкрийте сторінку:
   `https://github.com/MishchenkoDaniil/excel-archive-search-macos`
2. Натисніть `Code`
3. Натисніть `Download ZIP`
4. Дочекайтеся завантаження
5. Розпакуйте ZIP-файл подвійним кліком

### 2. Запустіть програму

1. Відкрийте папку `excel-archive-search-macos`
2. Знайдіть файл `bootstrap.command`
3. Натисніть на нього двічі

Якщо macOS не дає відкрити файл:

1. Відкрийте `System Settings -> Privacy & Security`
2. Дозвольте запуск
3. Запустіть `bootstrap.command` ще раз

### 3. Дочекайтеся відкриття програми

Під час першого запуску Mac:

- підготує все потрібне;
- встановить потрібні компоненти;
- відкриє програму в браузері.

Зазвичай адреса така:

```text
http://localhost:8501
```

## Як користуватися

1. Натисніть `Обрати у Finder`
2. Виберіть папку, де лежать ваші Excel-файли
3. Натисніть `Індексувати`
4. Дочекайтеся завершення
5. Введіть текст у блоці `Пошук`
6. Оберіть режим пошуку
7. Перегляньте результати

## Як запустити ще раз пізніше

Якщо програма вже була встановлена раніше, відкрийте `Terminal` і виконайте:

```bash
cd ~/Downloads/excel-archive-search-macos
./bootstrap_mac.sh
```

Якщо папка програми лежить не в `Downloads`, просто перейдіть у свою папку з програмою і виконайте:

```bash
./bootstrap_mac.sh
```

## Як зупинити програму

У вікні `Terminal`, де запущена програма, натисніть:

```text
Control + C
```

## Що вміє програма

- пошук за назвою файлу;
- пошук за текстом усередині Excel;
- показ папки, файлу, аркуша, рядка, колонки і комірки;
- оновлення індексу після змін у файлах;
- очищення застарілих записів;
- експорт результатів у Excel.

## Основні файли

- `bootstrap.command` — запуск подвійним кліком
- `bootstrap_mac.sh` — запуск через Terminal
- `app.py` — інтерфейс програми
